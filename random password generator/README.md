# Random Password Generator

A simple Python program that generates a random password using letters, digits, and symbols.

## Description
The program creates a password of fixed length (6 characters) using:
- Uppercase letters
- Lowercase letters
- Numbers
- Special symbols

Each character is randomly selected.

## Files
- main.py → contains password generation logic

## How to Run
Run the program using:

```bash
python main.py
```
## How It Works
- Define character pool (letters + digits + symbols)
- Loop for required password length
- Randomly pick characters
- Combine into final password
- Display result
