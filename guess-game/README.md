# Random Number Guessing Game

A simple Python console game where the computer generates a random number between 1 and 100 and the user tries to guess it.

## Files
- main.py → game logic

## How to Run
Run this command in terminal:

```bash
python main.py
```
## Rules
- Enter a number between 1 and 100
- Type Q to quit the game
- The program gives hints:
- Too small → increase your guess
- Too big → decrease your guess
- The game ends when the correct number is guessed or the user quits
